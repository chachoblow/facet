# Facet

A markdown editing experience designed around shared alignment — across humans, AI agents, and your future self.

> One document. Many faces. Same source.

Facet treats markdown files as the canonical artifacts that collaborators (devs, PMs, AI agents) align around. Different audiences need different surfaces; the underlying content stays in git.

## Surfaces

| Surface | Audience | Editing mode | Status |
|---|---|---|---|
| **Facet for VS Code** | Devs authoring | Hybrid live-preview | In development (v1) |
| **Facet Review** | All collaborators | Renderer with comment overlay | Future |
| **Facet Studio** | PMs authoring | True WYSIWYG | Future |
